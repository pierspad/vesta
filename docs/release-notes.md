
## Release Notes v0.10.0

### New Features

* **Complete UI Overhaul**: Redesigned the application interface with a cleaner structure, improved visual consistency and a simpler user experience across all sections.
* **Organized Settings Layout**: Reworked the settings experience with clearer categories and dedicated sections for each configuration area.

### Fixes

* **UI Stability Improvements**: Fixed multiple UI issues across windows and components.
* **Scrolling Behavior**: Fixed scrolling problems and improved navigation in larger layouts.
* **Flashcard Table Layout**: Fixed column alignment and improved flashcard table presentation.
* **Flashcard Interaction**: Fixed drag and drop behavior in the flashcard window.
* **Localization Fixes**: Improved translations and fixed inconsistencies across supported languages.

### Improvements

* **Simplified User Experience**: Reduced interface complexity and improved the organization of controls and options.
* **Window Layout Consistency**: Unified window decorations and added consistent header elements across the application.
* **UI Components Refresh**: Updated various UI elements with a more coherent design system.
* **Whisper Refactoring**: Improved the internal Whisper integration by making the architecture more modular and maintainable.
